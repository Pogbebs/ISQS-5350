Welcome to Our User Management System

This application is designed to demonstrate a basic user management system where users can create an account, log in, and view a list of users. The system is built using HTML, CSS, JavaScript, Python, and MySQL.

Features

  • User Registration: New users can sign up by providing a username and password.
  • User Login: Existing users can log in using their credentials.
  • User List Display: A list of all registered users can be viewed.

How It Works

Creating an Account

When a new user fills out the form on the 'Create Account' page and submits it, JavaScript sends the username and password to a Python script on the server. The script hashes the password for security, then stores the username and hashed password in a MySQL database.

Logging In

When a returning user enters their username and password on the 'Login' page, JavaScript sends these credentials to another Python script on the server. This script checks the database to see if an account with that username exists and compares the provided password with the stored hashed password using bcrypt.

Viewing User List

When visiting the 'View Users' page, the server-side Python script retrieves all registered users from the database and displays them. Passwords are not displayed for security reasons.

Security Measures

We take security seriously. Here are some of the measures in place:

  • Passwords are hashed using bcrypt before being stored in the database.
  • User inputs are sanitized to prevent SQL injection and XSS attacks.
  • Communication with the server should be done over HTTPS (not covered in this demo for simplicity).
